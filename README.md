# Assignment 2 Version Control Practice
---

## 程式內容

兩個功能分支的程式碼：

* `input.py`: 負責處理資料的接收與處理。
* `output.py`: 負責處理圓餅圖結果的輸出。

### 分支管理

* **`main`**: 主線分支，包含所有已完成、測試並上線的功能。
* **`feature-input`**: 輸入模組的開發分支 (已合併至 main)。
* **`feature-output`**: 輸出模組的開發分支 (已合併至 main)。
